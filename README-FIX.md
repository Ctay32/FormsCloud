## ⚡ QUICK START (3 MINUTES)

### Étape 1: Fix Supabase
1. Ouvrez: **`FIX-SUPABASE-MAINTENANT.sql`**
2. Copiez TOUT son contenu (Ctrl+A → Ctrl+C)
3. Allez à: https://supabase.com/dashboard
4. Sélectionnez "projet formcloud"
5. Cliquez "SQL Editor" → "New Query"
6. Collez (Ctrl+V)
7. Cliquez "RUN"
8. Attendez: `SUCCESS: All policies have been reset!`

### Étape 2: Redémarrer

## 🆘 AIDE D'URGENCE

### Problème lors de l'exécution du SQL?
→ Vérifiez que vous avez copié **TOUT** le contenu  
→ Regardez le message d'erreur  
→ Relisez l'étape 1

### Permission Denied?
→ Le SQL n'a pas been executed correctement  
→ Réessayez

### Les données n'apparaissent pas?
→ Actualisez la page Supabase (F5)  
→ ou redémarrez l'app Next.js

### Erreur JavaScript?
→ Redémarrez: Ctrl+C puis `npm run next-dev`
